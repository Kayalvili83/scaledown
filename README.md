Project Title: Loan Application Bot

Description: A chatbot that assists users with mortgage, auto, and personal loans. Uses ScaleDown to compress documentation and pre-qualify applicants quickly.

Features:

Loan comparison & eligibility calculator

Pre-qualification in under 2 minutes

Document collection & compression (80% reduction)

Application tracking & payment calculators

Interest rate monitoring

Tech Stack: Python, Flask/FastAPI, ScaleDown, GitHub Actions (for CI/CD), SQLite/MySQL

Installation:

git clone https://github.com/Kayalvili83/scaledown.git
cd scaledown
pip install -r requirements.txt
python main.py

Usage: Open the bot via localhost or your deployed URL, interact to check eligibility, compare loans, or upload documents.

Contributing: Open to suggestions, pull requests, and bug reports.
